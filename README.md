# iPAS AI應用規劃師 輔助小天使免費版

這是一個由 **Falo x Force 原力狐老師**整理的 iPAS AI應用規劃師題庫輔助工具。

目前版本：`v2.03`

更新時間：`update 2026/4/26`

GitHub Pages 入口：

```text
index.html
```

## 這個工具做什麼

v2.03 這次主要更新三件事：

1. 納入 2026 第 1 次初階官方考試題庫。
2. 增加初階考點分析，可比較兩次初階 A1 / A2 題型與官方考綱分類。
3. 加入更人性化的延伸互動設計，讓練習、模擬測驗、考後檢討與 PDF 留存更直覺。

- 題庫閱覽
- 模擬測驗
- 科目與年份組合出題
- 錯題回顧
- 測驗結果列印 / 存為 PDF
- 支援 2025 與 2026 題型組合練習
- 初階 A1 / A2 兩次考題官方考綱分類比較
- 初階官方考綱對照與互動 SVG 分層檢視

## v2.03 本版重點

```text
update 2026/4/26｜免費版 v2.03｜新增初階考點分析：可比較兩次初階 A1 / A2 題型與官方考綱分類。中階分類分析將於 v2.04 補齊；AI威力版功能更齊全。
```

- 2026 第 1 次初階官方考試題庫已納入，可與既有初階題庫混合練習。
- 初階考點分析新增兩次考題比較與官方考綱對照，可觀察 A1 / A2 命題變化。
- 首頁導覽順序改為「初階考點分析 → 官方題庫閱覽 → 模擬測驗」，三個入口使用不同顏色與外框辨識。
- 初階考點分析改為左側圖片摘要、右側互動分析；原始 PNG 圖片保留於 `assets/analysis/`。
- 官方考綱對照新增互動 SVG，可切換 A1 / A2 / 全部，也可切換 L1 / L2 / L3 層級並展開下一層。
- 模擬測驗新增兩種答案顯示方式：
  - 練習模式：每題送出後立即顯示答案。
  - 模擬考模式：全部考完後才一次顯示作答檢討。
- 模擬測驗新增答題計時器；答題中計時，查看解答與結果頁暫停，結果頁顯示總答題時間、平均每題時間與每題時間佔比。
- 保留錯題 CSV，但改成人可閱讀的錯題檢討格式，檔案開頭含版權宣告、網站網址、版本與輸出時間，不作為題庫匯入格式。
- PDF 列印內容加入列印時間、答題時間、版權與使用聲明，並使用淡色 `Falo x Force` 浮水印。
