# 版本說明：v2.03

更新時間：update 2026/4/26

這份版本延續 v2.01 免費版，主要納入 2026 第 1 次初階官方考試題庫，新增「初階考點分析」分區，並補強更人性化的練習、測驗與考後檢討互動。

```text
update 2026/4/26｜免費版 v2.03｜新增初階考點分析：可比較兩次初階 A1 / A2 題型與官方考綱分類。中階分類分析將於 v2.04 補齊；AI威力版功能更齊全。
```

## 版本定位

v2.03 的目標是讓「iPAS AI應用規劃師 輔助小天使免費版」從題庫與模擬測驗工具，升級成可以觀察兩次初階考點變化的備考輔助工具。

本版主軸：

1. 更新初階 2026 第 1 次官方考試題庫。
2. 增加考點分析，支援兩次初階 A1 / A2 與官方考綱分類比較。
3. 增加更人性化的延伸互動設計，讓使用者可以更自然地練習、測驗、檢討與保存結果。

目前已完成：

- 題庫資料來源切換到 v2.03 多分頁 Google Sheet
- 題庫筆數維持 350 題
- 已納入 2026 第 1 次初階 A1 / A2 官方考試題庫
- 測驗題數上限改為依實際題庫總數更新
- 畫面已標示 2026 第 1 次初階考試題型已納入
- 題庫閱覽與模擬測驗的選擇介面已改為單一組勾選按鈕
- 每個按鈕直接代表「科目 + 年份/梯次」，例如 A1｜2026 第1次初階
- 預設不勾選任何題型，並提供全部選擇 / 全部取消
- 混合題型測驗會盡量平均分配各題型題數
- 結果頁新增題型分布與答題表
- 題目與作答回饋可顯示 A1 / A2 官方考綱分類
- 新增「初階考點分析」分區
- 「初階考點分析」內含「兩次考題比較」與「官方考綱對照」
- 官方考綱對照支援互動 SVG，可切換 A1 / A2 / 全部與 L1 / L2 / L3 層級
- 測驗結果新增已 mapping 題目的考綱弱點摘要
- 模擬測驗新增練習模式與模擬考模式，可選擇每題顯示答案或考後一次檢討
- 模擬測驗新增答題計時器，答題中計時、查看解答與結果頁暫停
- 測驗結果新增每題作答時間分析，同時顯示絕對時間與佔總答題時間百分比
- 測驗結果可列印 / 存為 PDF，含列印時間、版權聲明與 Falo x Force 浮水印
- 保留錯題 CSV 輸出，但採人可閱讀的錯題檢討格式，檔案開頭含版權宣告、網站網址、版本與輸出時間
- Header 已加入 Falo x Force 原力狐老師 LinkedIn 連結
- 已加入 SEO、Open Graph、Twitter Card、Schema.org 與 GEO metadata
- 已標註「小天使免費版 v2.03，AI威力版功能更齊全」

## 目前欄位

`題庫` 分頁仍使用 v1.01 相容的 10 個核心欄位：

| 欄位 | 用途 |
| --- | --- |
| `編號` | 判斷是否為有效題目 |
| `分類` | 科目分類，例如 A1、A2、B1、B2、B3 |
| `題號` | 題目排序 |
| `文件名稱` | 題目來源檔名，可用來推斷考試梯次 |
| `題目` | 題幹 |
| `選項1` | 第一個選項 |
| `選項2` | 第二個選項 |
| `選項3` | 第三個選項 |
| `選項4` | 第四個選項 |
| `正確答案` | 正確選項，格式為 1、2、3、4 |

另外新增：

| 分頁 | 用途 |
| --- | --- |
| `official_syllabus_topics` | 官方簡章考綱分類，整理到 L1 / L2 / L3 |
| `question_topic_mapping` | A1 / A2 題目對應官方分類，含 confidence 與 needs_review |
| `summary` | 初階分類統計與品質觀察 |

## 目前新版邏輯

目前程式已從「只依單一科目出題」升級成：

```text
勾選一個或多個科目年份組合 → 題庫閱覽 / 模擬測驗 → 組合出題
```

並新增：

```text
初階考點分析 → 兩次考題比較 / 官方考綱對照
```

下一步預計新增能力：

- v2.04 補齊 B1 / B2 / B3 中階分類 mapping
- 將分類分析擴充到全考科
- 未來若 Google Sheet 新增 `詳解`、`難度`、`章節` 等欄位，程式可直接呈現在回饋與篩選流程中
